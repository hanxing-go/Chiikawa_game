## Chiikwa讨伐大作战
1. 窗口的创建
    - 设置宽高
      - 设置为横版游戏
    - 设置Icon
    - 设置图片背景
      - 利用PPT和画图，对图片进行裁剪，消除，改变尺寸
      - 这里遇到了相当大的困难，JFrame重写paint方法不管用
    - 设置游戏开始界面


2. 设置游戏的开始，暂停，退出
   - 开始：按空格键或者鼠标指针点击窗口任意位置开始
   - 暂停：按空格键暂停
   - 退出：按下exit键退出，点击窗口按钮退出


3. 滚轮切换武器或者装备之类的

4. 游戏物体
   - 公有属性
      - 物体所在的x轴
      - 物体所在的y轴
      - 物体本身的图片
      - 物体的宽度
      - 物体的高度
      - 物体的移动速度
